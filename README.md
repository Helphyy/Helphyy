## about

I build and run infrastructure, automate what can be automated, then attack it to
check that it holds. Virtualization, distributed storage, and the tooling I write
along the way when it does not already exist.

Currently at **Green IT Solutions**, working across infrastructure, DevOps and
offensive security.
